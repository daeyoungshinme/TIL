# TIL(Today I Learned)
>## 개요
오늘 하루 공부한 내용을 정리해서 기록하는 목적으로 작성하자.
>## 작성법 
1. 파일명은 날짜.md(ex.20220906.md) 규칙으로 작성한다.
2. 마크다운 문법(정리해서 링크)을 통해 작성한다.
3. 공부한 내용은 반드시 나의 생각 혹은 소스를 통해 아웃풋 형태로 작성한다.
4. TIL작성은 메모처럼 작성후 각 주간/월간으로 각 카테고리에 분류 및 추가 정리한다.
